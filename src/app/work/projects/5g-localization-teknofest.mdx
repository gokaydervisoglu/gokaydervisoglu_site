## Overview

Competed in the TEKNOFEST 5G Localization competition as a member of the AI2X Team, achieving 4th place nationally. Developed a Kalman filter–based hybrid localization system using 5G RSRP and SINR data, achieving positioning errors of approximately 5–150 meters in test scenarios.

## Key Features

- **Kalman-based hybrid localization** using 5G RSRP and SINR measurements
- **Noise reduction** and temporal smoothing of signal data
- **National-level ranking** in the TEKNOFEST 5G Localization category

## Technologies Used

- **Python**
- **Pandas / NumPy**
